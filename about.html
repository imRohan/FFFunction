About FFFUNCTIONS


Functions are all around us. They play a major role in computer science, math and literally give meaning to objects. The concept of a function was the basis for the development of this application. A function in computer science terms is a piece of code that can be called upon at will. The idea of a block of code, which can be repeated at will, was fascinating, and led me to the wonderful world of fractal art.

From Wikipedia: '' Random fractals have been used to describe/create many highly irregular real-world objects. A limitation of modeling fractals is that resemblance of a fractal model to a natural phenomenon does not prove that the phenomenon being modeled is formed by a process similar to the modeling algorithm.''

This was the inspiration for the application FFFUNCTIONS. The name, meaning 'Fractals from Functions' is the essence of what this application does. By using simple functions, bezier curves are drawn on a canvas repeatedly. By drawing several curves each second, the growth of a fractal will be observed by the user.



Using touch gestures such as Touch, Touch and Hold, Swipe Left and Swipe Right, the user will be able to interact with the piece. Since this application will run on both desktops as well as mobile devices, the Swipe Up and Swipe Down touch gestures were not used due to scrolling issues. When the user touches the screen (TouchStart) the drawing function will be called. As soon as the user removes the finger (TouchEnd) the drawing function will be terminated, thus stopping the growth of the fractal. By doing this the user is forced to interact with the device in order to create fractal art. By swiping left or right, the color of the fractals will change. By having several color pallets, the user will be able to create fractals which seem to have more depth.
